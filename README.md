# By Milia — сайт (MVP)

Статический сайт с каталогом (4 расцветки × 6 размеров), корзиной и заявкой в WhatsApp + email.

**Сайт:** https://tolkachev14881488-beep.github.io/bymilia/

## Запуск локально

Нужен локальный сервер (модули ES6 не работают с `file://`):

```bash
npx --yes serve .
```

Откройте адрес из терминала (обычно http://localhost:3000).

## Админка (CMS)

**URL:** `/admin/` (локально: `http://localhost:3000/admin/`)

- **Пароль по умолчанию:** `bymilia2026` (смените в разделе «Настройки»)
- **Товары:** цены, описания, фото (URL), скрытие из каталога
- **Главная и страницы:** тексты блоков и HTML-страниц
- **Контакты и доставка**

Данные сайта лежат в **`data/site.json`** и **`data/products.json`**.

### Публикация на GitHub Pages

1. Создайте [Personal Access Token](https://github.com/settings/tokens) с правом **repo**
2. В админке → «Публикация» → вставьте токен, владелец `tolkachev14881488-beep`, репозиторий `bymilia`
3. Нажмите **«Опубликовать на сайт»** — через 1–3 минуты сайт обновится

Альтернатива: **«Скачать JSON»** → положите файлы в `data/` и сделайте `git push`.

## Что настроить без админки

Файлы **`data/site.json`** и **`data/products.json`** (или через админку).

Положите в **`assets/`**:

- `logo.png` — уже есть
- фото товаров — URL в карточке товара в админке
- `pricelist.pdf` — для страницы «Опт»

## Структура

- `index.html` — главная
- `catalog.html`, `product.html`, `cart.html`
- `pages/` — информационные страницы
- `css/styles.css` — стили (акцент: оранжевый `#ff7f00`, тёплая палитра)

## Заказ

Корзина хранится в `localStorage`. По кнопке «Отправить» открываются WhatsApp и почтовый клиент с текстом заказа.

Для продакшена без `mailto` можно подключить Formspree, Telegram Bot API или простой backend.
