Есть черновик —

Выберите товар слева, измените название, цену и фото. «Сохранить и опубликовать» — сразу на сайт.

Для разработчика: импорт с Wildberries

Тексты блоков на главной странице. После правок — «Сохранить и опубликовать».

Тексты страниц «Опт», «Доставка» и др. Можно использовать простые HTML-теги.

Телефон, WhatsApp, адрес и название бренда — отображаются в шапке и подвале сайта.

Настройка один раз

  1. Откройте админку на любом устройстве и войдите по паролю.
  2. Меняйте товары, тексты и контакты.
  3. Нажимайте «Опубликовать всё» — изменения отправятся прямо на by-milia.by.

Для разработчика: резервный режим через GitHub

Обычная публикация идёт напрямую на by-milia.by. Локальный режим нужен только как запасной: start-cms-publish.bat.
